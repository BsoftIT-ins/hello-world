PHP Loops
----------
There are 4 types of LOOPS in PHP
1. While Loops
2. Do while loops
3. For Loop
4. Foreach loop

While Loop
------------
Syntax >>
while (condition is true){
  code to be executed;
}

Example:
<?php
$x = 1;
while ($x <= 5){
echo "The Number is : $x <br>";
  $x++;
}
?>
